# Kanban Task Board
Name : Hebatulla Omer Abdulla ALi 

Matric: A24CS4014 

A vanilla JavaScript Kanban board built for my Cross Platform App Development course.

## Features
- Add, edit, and delete tasks across To Do, In Progress, and Done columns
- Filter tasks by priority (High / Medium /Low)
- Double-click any task title to rename it inline
- Staggered fade-out animation when clearing the Done column
- Built with pure HTML, CSS, and JavaScript 

## How to run
Download the files and open `index.html` in any browser.

## Concepts i practised in this lab task
- DOM manipulation with createElement, setAttribute, appendChild
- Event delegation (one listener handles all card buttons)
- CSS class toggling for filtering and animations
